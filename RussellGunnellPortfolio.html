Kiosk Project


-The total cost of each item purchased is calculated.
-The user can input individual bill/coin values until the full price is paid.
-The user can also enter credit/debit card number instead
-The program keeps an inventory of every bill and coin in the kiosk.
-The user is refunded if the kiosk does not have the sufficient change required.
-Another program starts when prompted by the current program.
-The second program creates a TXT file named after the current date, or appends this file if it already exists.
-Details of the transaction are listed within the TXT file.



Ascii Art Generator


-The program loads an image of the user's choice.
-The image is assigned to a bitmap, where its pixel colors are converted to grayscale
-The average grayscale value in a 5 by 10 area of pixels is calculated for every 5 by 10 space in the image.
-The grayscale value from each average is converted into a pair of ascii characters.
-The ascii characters are added to a string variable that eventually becomes the final product.
-Line breaks are inserted into the string variable whenever the edge of the bitmap is detected.



Encoder Decoder Project


-The Encoder program reads data from a PPM file of the user's choice. It can be a P3 or P6 type.
-The A bitmap is created based on the size and color values in the PPM file, and it is displayed as an image.
-A modified version of the previous bitmap is created with the user's desired message encoded inside as decimal values.
-The modified bitmap is displayed as another image beneath the previous one.
-The modified image is saved as a PPM file.
-On the separate Decoder program, the user chooses a modified PPM file to decode.
-The file is displayed as an image like before, and the encoded message is displayed to the user.
